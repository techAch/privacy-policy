Privacy Policy

Last updated: June 19, 2026

This Privacy Policy explains how this application handles user data.

Data Collection

This application does not collect, store, transmit, or share personal data.

Local Storage

Trip history, expenses, calculations, and app settings are stored locally on the user's device only.

No user data is uploaded to external servers.

Analytics

This application doesn't use any analytics services. No data is sent to third-party analytics providers.

Third-Party Services

This application does not use advertising services.

Offline Usage

The application is designed to function offline and does not require an internet connection for core functionality.

Data Deletion

Users may remove locally stored data by clearing the application data or uninstalling the application.

Contact

If you have questions about this Privacy Policy, you may contact the developer.
