Politique de confidentialité

Dernière mise à jour : 1 juillet 2026

Cette politique de confidentialité explique comment cette application traite les données des utilisateurs, y compris les informations de compte lorsque vous vous connectez avec Google.

Collecte de données

Lorsque vous vous connectez avec Google, l'application utilise Firebase Authentication et Google Sign-In pour authentifier votre compte. Cela peut impliquer la collecte et le traitement des informations de profil de votre compte Google, telles que votre nom, votre adresse e-mail et un identifiant utilisateur unique, ainsi que des informations liées à l'authentification nécessaires pour vous maintenir connecté.

Les informations de connexion sont stockées de manière sécurisée via Firebase Authentication afin que l'application puisse reconnaître votre compte et maintenir votre session de connexion.

Stockage local

L'historique des trajets, les dépenses, les calculs et les paramètres de l'application sont stockés uniquement localement sur l'appareil de l'utilisateur.

Les informations de connexion au compte sont gérées via Firebase Authentication plutôt que d'être stockées uniquement sur l'appareil.

Analytique

Cette application n'utilise aucun service d'analytique. Aucune donnée n'est envoyée à des fournisseurs d'analytique tiers.

Services tiers

Cette application utilise Google Sign-In et Firebase Authentication pour fournir l'accès au compte. Google et Firebase peuvent recevoir les informations nécessaires pour authentifier votre compte et fournir ces services.

Utilisation hors ligne

L'application est conçue pour fonctionner hors ligne pour les fonctionnalités principales, bien que la connexion au compte et la synchronisation puissent nécessiter une connexion Internet.

Suppression des données

Les utilisateurs peuvent supprimer les données stockées localement en effaçant les données de l'application ou en désinstallant l'application. Pour demander la suppression des données liées au compte gérées via Firebase, veuillez contacter le développeur.

Contact

Si vous avez des questions concernant cette politique de confidentialité, vous pouvez contacter le développeur.
