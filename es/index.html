Política de privacidad

Última actualización: 19 de junio de 2026

Esta política de privacidad explica cómo esta aplicación gestiona los datos de los usuarios.

Recopilación de datos

Esta aplicación no recopila, almacena, transmite ni comparte datos personales.

Almacenamiento local

El historial de viajes, los gastos, los cálculos y la configuración de la aplicación se almacenan únicamente de forma local en el dispositivo del usuario.

No se suben datos de usuario a servidores externos.

Analítica

Esta aplicación no utiliza servicios de analítica. No se envían datos a proveedores de analítica de terceros.

Servicios de terceros

Esta aplicación no utiliza servicios publicitarios.

Uso sin conexión

La aplicación está diseñada para funcionar sin conexión y no requiere acceso a Internet para su funcionalidad principal.

Eliminación de datos

Los usuarios pueden eliminar los datos almacenados localmente borrando los datos de la aplicación o desinstalándola.

Contacto

Si tiene preguntas sobre esta política de privacidad, puede contactar al desarrollador.
